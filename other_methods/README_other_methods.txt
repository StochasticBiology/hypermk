Here we try to run the phylogenetic and cross-sectional data with phytools and geiger.

Files:
- phylog-examples.R : try to run single.

- cross-sectional-examples.R: cross.sectional.

- datasets-phylog-cross-sect.R: how to create an RData that pulls the datasets from mk-collection.R to make it easier to run other methods on the exact same data. (Not used in cross-sectional-examples.R, but used in phylog-examples.R)
